# Changelog

## [1.0.0] - {PR_MERGE_DATE}

### Added

- Initial version
- Shows the current date and time as a live subtitle in Raycast root search, refreshing every minute
- Copies the formatted value to the clipboard when the command is run
- Configurable date format supporting the full strftime directive set (via the [strftime](https://github.com/samsonjs/strftime) library)
- Timezone support: pick any IANA timezone or a fixed UTC offset
